# saltqlab

## How to compile:
```
    g++ saltqlab.cpp -o saltqlab -Wall -Wextra -O2
```



Command line arguments:

```
   * Usage: "saltqlab [ -h | -t | -hl ]"
   * Invalid args                    Displays help.
   * -h                              Displays help.
   * -t                              Testing mode. Not implemented.
   * -hl                             Displays the license (GPL-3).
```
